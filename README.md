# EMRS JSA Typing Test Practice

EMRS JSA Typing Test Practice tool is designed for candidates preparing for Eklavya Model Residential School Junior Secretariat Assistant typing examinations.

## Features

- EMRS JSA typing practice
- English and Hindi typing test
- TCS iON style interface
- Timer based typing practice
- Gross speed and net speed result
- Accuracy and error tracking
- Mobile responsive layout

## Typing Rules

- Exam: EMRS Junior Secretariat Assistant
- Typing Language: English & Hindi
- English Speed Target: 35 WPM
- Hindi Speed Target: 30 WPM
- Backspace: Allowed in practice
- Recommended Accuracy: 90%+
- Practice Modes: 2 Min, 5 Min, 7 Min
